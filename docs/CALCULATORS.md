**Calculation Formula:**

```
bytes_per_pixel = bit_depth / 8
image_size_bytes = width ร height ร bytes_per_pixel ร channels ร z_slices ร time_points
raw_total_bytes = image_size_bytes ร image_count
compressed_bytes = raw_total_bytes ร compression_ratio  # Default 0.1 (10x compression)
with_buffer = compressed_bytes ร 1.2
with_processing = with_buffer ร (processing_enabled ? 2 : 1)
```

**Output Format:**
- Primary: "X.X TB"
- With growth: "After N years: X.X TB"
- Recommended: "X TB (with buffer)"

## Configuration

All calculators can be customized in `config/calculators.yaml`:

```yaml
calculators:
  # Enable/disable calculators
  storage:
    - microscopy
    - genomics
    - video
    - simulation-output
    - general

  compute:
    - hpc-batch
    - genomics-pipelines
    - simulations

  gpu:
    - ml-training
    - ml-inference

  # Global settings
  global:
    buffer_multiplier: 1.2      # Add 20% buffer to all estimates
    show_calculation: true      # Show math breakdown
    show_cost_estimate: true    # Show dollar amounts

  # Per-calculator customization
  microscopy:
    presets:
      - id: confocal-core
        label: "Confocal Core"
        resolution: "2048x2048"
        bit_depth: 16
        channels: 4
      - id: light-sheet
        label: "Light Sheet"
        resolution: "2048x2048"
        bit_depth: 16
        channels: 2
        z_slices: 200

  genomics:
    # Institution-specific pipeline benchmarks
    pipelines:
      wgs_alignment:
        su_per_sample: 300
        note: "BWA-MEM2 on our cluster"

  ml_training:
    # Available GPU types at this institution
    gpus:
      - id: v100
        name: "NVIDIA V100 (16GB)"
        available: true
      - id: a100-40
        name: "NVIDIA A100 (40GB)"
        available: true
      - id: a100-80
        name: "NVIDIA A100 (80GB)"
        available: false
        note: "Coming Q2 2024"
```
